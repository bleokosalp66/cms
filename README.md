# cms

## Project Overview
This project is a simple content management system (CMS) example that demonstrates the basic functionalities of a CMS.

## Features
- User authentication.
- Post creation and management.
- Simple admin interface.

## Technologies Used
- PHP
- MySQL
- HTML/CSS

## Usage
1. Clone the repository.
2. Set up the database.
3. Run the application.

## Future Plans
- Add more features like comments and tags.
- Improve the user interface.
